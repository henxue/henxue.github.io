帝国CMS修改 morepic 前端显示样式

  • henxue 发布于 4小时前
  • 分类：帝国教程
  • 阅读()
  • 评论

[!--morepic--]是图片集字段，有人说这是图片系统固有的字段，其实这是错的，帝国所有的系统模型都是可以变化的，并不是官方默认给你几个、给你什么模型就是什么模型，这也就是帝国CMS的灵活性所在。[!--morepic--]可以用在任何你想要它出现的任何系统模型里，那么怎样修改[!--morepic--]的外观呢？这也是帝国模板比较难也是比较变态的一个地方，没有把模板和程序分开，所以我们是要修改程序的。

打开e/class/functions.php这个文件。搜索//返回图片集html代，默认的大概是在2763行，如果你都没有修改过这个文件的话。
我们先来看一下这段代码，代码很多，要有耐心去看！

//返回图片集html代码
function ReturnMorepicpathHtml($add){
global $public_r,$fun_r;
if(empty($add[morepic]))
{
return ”;
}
$line=$add[num]?$add[num]:1;//每行显示
$picpath=”;
$path_r=explode(“rn”,$add[morepic]);
for($pj=0;$pj<count($path_r);$pj++)
{
$p=$pj+1;
if(($p-1)%$line==0||$p==1)
{
$picpath.=’<tr>‘;
}
$showdown_r=explode(‘::::::’,$path_r[$pj]);
//显示图片名称
$name=”;
if(!empty($showdown_r[2]))
{
$name=”<br><span style=’line-height=18pt’>”.$showdown_r[2].”</span>“;
}
$width=$add[width]?” width=’”.$add[width].”‘”:”;//宽度
$height=$add[height]?” height=’”.$add[height].”‘”:”;//高度
$picpath.=”<td align=center><a href=’”.$public_r[newsurl].”e/ViewImg/index.html?url=”.$showdown_r[1].”‘ target=_blank><img src=’”.$showdown_r[0].”‘”.$width.$height.” border=0>”.$name.”</a></td>”;
//分割，其实这一段是主要的显示区域，也就是说，我们要改变前端显示的话，一般就是改变这一段
//”.$public_r[newsurl].”e/ViewImg/index.html?url=”.$showdown_r[1].” 这一段是帝国默认的功能，在新的页面中打开这张图片，还可以缩放来着
//”.$showdown_r[0].” 懂html的都懂，这是图片url
if($p%$line==0)
{
$picpath.=’</tr>‘;
}
}
if($p<>0)
{
$table=”<table width=’100%’ border=0 cellpadding=4 cellspacing=4>“;
$table1=”</table>“;
$ys=$line-$p%$line;
$dotr=0;
for($j=0;$j<$ys&&$ys!=$line;$j++)
{
$dotr=1;
$picpath.=’<td></td>‘;
}
if($dotr==1)
{
$picpath.=’</tr>‘;
}
}
$value=$table.$picpath.$table1;
return $value;
}

其实说修改前端显示外观，就是把帝国惯用的table改成我们自己想要的就可以啦，请注意上面代码中红色的代码，下面是我自己实现的一个样式：
